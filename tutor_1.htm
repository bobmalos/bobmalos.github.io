This tutorial steps you through the Forms that create an SQL (Structured Query Language) query for you.
  • You may fill in the Boxes and Lists but you will not be changing the results.
  • If it looks like a List is empty, scroll-down to see what has been chosen.

If You Choose To Build A Query You Will See Five Forms After This One:

  • Form 2 - Choose the Database(s)
  • Form 3 - Choose the Table(s)
  • Form 4 - Choose the Column(s)
  • Form 5 - Choose the Condition(s)
  • Form 6 - Choose the Order (if any) and Execute the query

The results can be displayed on your screen and/or written to a file in one of several formats (Excel, Lotus, DBF, or ASCII-Delimited-Text).

Choose A Querying Method:

  • Build A Query (with 'Point and Click')
  • Free-Form Query
  • Choose A Saved Query
  • Delete One Or More Saved Queries

It will take a few seconds for your ID and Password to be Verified.
