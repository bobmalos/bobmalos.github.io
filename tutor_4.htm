Choose The Column(s) You Want In Your Query Results:

If the Column Name is Underlined, you can link to its Data Dictionary entry.

Use the smaller CheckBox to choose a column WITHOUT an order
OR use the Input Box to choose a column WITH an order (1 - 99).
You do not have to Order each Column.

You may also put 's' or 'c' in the Input Box for a SUM or COUNT Function. (both 's' and '1s' would be OK) If you do this, a GROUP-BY clause will be generated for you. It is SUGGESTED (but not required) that if you use Function(s), you specify a numerical order for each Column.
